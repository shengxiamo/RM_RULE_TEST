# 基于题库的RoboMaster规则测评答案检索脚本

## 使用方法
### 1.保存试题
在浏览器界面cltr+s保存html文件

![img0.png](data/img0.png)

## 2.构建项目
把“完整题库 最低99最高100.xlsx”文件移到“main.py”、“RMUC 2025规则测评.html”同一目录下。

![img1.png](/data/img1.png)

3.安装依赖
```bash
pip install -r requirement.txt
```
